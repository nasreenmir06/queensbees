About the Project

In this project, we will dig into the software architecture of ScummVM. This open-source website is a virtual machine that allows you to run old adventure and role-playing games like Blade Runner, Monkey Island and many more when you have the correct data files. The project is broken into 3 sections:

  • Assignment 1 is where we analyze the documentation and conceptual architecture of ScummVM.
  • Assignment 2 will look into its source code and report our findings on the concrete architecture.
  • Assignment 3 is when we propose ideas of how the software architecture can be improved upon.
